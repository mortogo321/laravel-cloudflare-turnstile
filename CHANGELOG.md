# Changelog

All notable changes to `laravel-cloudflare-turnstile` will be documented in this file.

## v1.0.0 - 2025-10-10

### Added
- Initial release
- Cloudflare Turnstile integration with invisible mode by default
- Auto-disable submit buttons until verification complete
- Show refresh button on verification failure
- Support multiple validation methods (rule class, string rule, manual)
- Full Laravel integration with service provider and Blade component
- Comprehensive documentation and usage examples
- Support for Laravel 10.x and 11.x
- Support for PHP 8.1+
